블로그에 댓글 기능 추가하기

블로그를 운영하다 보면 운영자와 사용자간의 소통이 필요한데 이를 위해 댓글 기능들을 넣는다.
그 중 GitHub blog에 손쉽게 추가할 수 있는 Disqus 댓글 기능을 이용하여 댓글 기능을 넣어볼려고 한다.

1. DISQUS 홈페이지를 접속하여 회원가입을 한다.

먼저 Disqus 댓글 기능을 사용할려면 DISQUS의 회원이 되어야 한다.

아래의 캡처 화면에서 Get Started 버튼을 클릭하여 회원가입을 진행한다.

2. 회원가입이 완료되면 아래와 같은 화면이 나오는데 두 번째 이미지를 클릭한다.

3. 클릭 시 댓글 기능을 사용할 Website url, 카테고리, 언어를 설정한 후 Site를 생성한다.

4. 생성하면 아래와 같은 Select a plan 라는 탭이 활성화 되어 있는데 세 가지 중 한 가지를 선택하여 Subscribe Now를 클릭한다.

저 같이 많은 사람들이 사용안하는 블로그라면 Basic을 선택합니다. (슬프네요)

5. 구독을 선택하면 여러가지 platform들이 나오는데 자신의 웹사이트의 platform을 클릭하여 진행하면된다.

만약 platform들 중 자신이 해당하는 platform이 없다면 아래에 버튼을 클릭하여 진행한다.
